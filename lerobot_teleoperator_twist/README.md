# lerobot_teleoperator_ros

Teleoperators for lerobot

This is a copy from: [lerobot_teleoperator_devices](https://github.com/ycheng517/lerobot-ros/tree/main/lerobot_teleoperator_devices) by Yicheng Cheng, which is a ROS package for teleoperation of lerobot using various input devices. The original package includes teleoperators for keyboard, joystick, and gamepad.
